# Markdown快速入门

## 基本

粗体用** **围住

**这是粗体**

斜体用* *围住

*这是斜体*

### 分割线

分割线是---
---

引言是>

> 引言部分

无序列表是- 

### 无序列表

- 第一行
- 第二行

---

有序列表是1.

### 有序列表

1. 第一行
2. 第二行

---

行内代码块用反引号 ` 围住

### 行内代码块

这是一行中的`代码块`，用反引号来围住。

---

代码块用 ```  围起来

### 代码块

``` Java
public class Demo{
        public static void main(String[] args) {
        System.out.println("Hello Markdown");
    }
}
```
